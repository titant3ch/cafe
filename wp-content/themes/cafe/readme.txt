== Installation ==

Manual installation:

1. Upload the `Cafe` folder to the `/wp-content/themes/` directory

Installation using "Add New Theme"

1. From your Admin UI (Dashboard), use the menu to select Themes -> Add New
2. Search for ‘Cafe’
3. Click the 'Install' button to open the theme's repository listing
4. Click the 'Install' button

Activation and Use

1. Activate the Theme through the 'Themes' menu in WordPress

== Theme Notes ==

= Post Thumbnail Functionality =

For the post thumbnails we recommend to use the image 752 px. width. 

= Post Formats: Standard, Aside, Quote, Status  =

Cafe Wordpress theme supports only these post formats, and displays them in a certain way. Other post formats will be displayed as a regular posts.

= Widgets =

The Theme includes two custom widgets except the defaults:
- Recent comments with excerpt (displays a short auto-generated comment excerpt)
- Recent posts with author and category

== Frequently Asked Questions ==

= What image size should I use for the Featured image? =

If the width of the image gallery is 752 px., you should use the of images 752 px. width, otherwise they might be scaled and/or cropped from the sides. 

= How are the Posts and Pages with Featured images are displayed? =

Posts and Pages where you have added the Featured images are displayed in certain way: the title and metadata (post or page date and post category) are displayed in a 'black-box' that is laid on the featured image and sticks to its bottom.

= How are Posts and Pages with Featured images and comments are displayed? =

Posts and Pages where you have added the Featured images, comment and/or any other content would display the comment (either user-specified or auto-generated) under the post or page title-and-meta area in case you put a mouse over the featured image. The post or page title-and-meta information will slide over to make room for a comment.

= How can I change the language from English into Russian? =

In the main catalog of your WordPress site open the file wp-config.php, and change define( ‘WPLANG’, '' ); → define( ‘WPLANG’, ‘ru_RU’ );

= Does the theme support widgets? =

Yes, this theme includes widgets. The following widgets are shown by default in a sidebar: search, recent posts, recent comments. By defaults, the footer displays recent posts, recent comments, year archives and menu.
